Pengenalan HTML

Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan merancang halaman web. HTML digunakan untuk menyusun struktur konten di internet.

Manfaat HTML

  • Membuat struktur dasar halaman web.
  • Mendukung multimedia seperti gambar, video, dan audio.
  • Memungkinkan penggunaan tautan (link) untuk navigasi antar halaman.
  • Memudahkan pengembangan web dengan berbagai framework dan teknologi lainnya.

Fungsi HTML

  • Menentukan elemen-elemen dalam sebuah halaman web, seperti judul, paragraf, gambar, dan tabel.
  • Menyediakan struktur yang jelas untuk konten yang ditampilkan di browser.
  • Memungkinkan pengembang untuk menambahkan atribut dan gaya pada elemen-elemen yang ada.

Kelebihan HTML

  • Mudah dipelajari dan digunakan, bahkan oleh pemula.
  • Kompatibel dengan berbagai browser dan platform.
  • HTML bersifat open-source dan gratis untuk digunakan.
  • Didukung oleh banyak alat dan editor yang memudahkan pengembangan.

Kekurangan HTML

  • HTML tidak dapat menangani logika pemrograman, seperti pengolahan data atau interaksi dinamis.
  • Kurangnya kontrol atas tampilan dan desain yang kompleks tanpa menggunakan CSS.
  • HTML tidak mendukung pengolahan data secara langsung, memerlukan bahasa pemrograman lain seperti JavaScript.

Contoh Kode HTML

        
            <!DOCTYPE html>
            <html lang="id">
            <head>
                <meta charset="UTF-8">
                <title>Contoh Halaman HTML</title>
            </head>
            <body>
                <h1>Selamat Datang di Halaman HTML</h1>
                <p>Ini adalah contoh paragraf dalam HTML.</p>
                <a href="https://www.example.com">Kunjungi Example.com</a>
            </body>
            </html>
